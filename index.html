About VerifyAI

VerifyAI brings together programming language theory, formal methods, and the latest LLM techniques to make formal verification practical for real-world codebases. We craft toolchains and workflows that deliver provable guarantees without slowing product velocity.

Our approach

  1. Compile Python Code in Dafny Code: Convert commonly used programming languages into programming languages that can be verified
  2. Use LLMs to Add Annotations: Use finetuned LLMs to add annotations to the Dafny code
  3. Use Dafny Methods to Verify Code: Dafny verifies your code
